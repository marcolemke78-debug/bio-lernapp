# Biologie-Lernprogramm: KA 2 (Klasse 8)

Interaktives Lernprogramm fuer die Biologie-Klassenarbeit Nr. 2 (Klasse 8, Realschule BW, M-Niveau).

**Themen:** Sinnesorgan Haut (Aufbau & Funktionen) · Vom Reiz zur Reaktion (Reiz-Reaktions-Kette & EVA-Prinzip) ·
Die Zelle (Bestandteile, Tier- und Pflanzenzelle im Vergleich) · Infektionskrankheiten (Bakterien & Viren).

8 Lektionen in 4 Modulen + Pruefungssimulation, jede mit Erklaerung, Beispiel und interaktiven Uebungen.
Reines HTML/CSS/JS, offline-faehig, Fortschritt im Browser gespeichert (localStorage). Eigene SVG-Grafiken,
keine urheberrechtlich geschuetzten Buchbilder.

Zusaetzlich zum Ausdrucken (im Browser Strg/Cmd + P):
- **Lernzettel** (`pdf/lernzettel.html`) - zweispaltiger Spickzettel
- **Karteikarten** (`pdf/karteikarten.html`) - zum Ausschneiden und Falten

Einfach `index.html` im Browser oeffnen.
